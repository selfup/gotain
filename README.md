# Gotain yourself

1. ssh into VM (can be run locally if on linux)
2. Clone repo: `git clone https://github.com/selfup/gotain`
3. cd into repo
4. `./scripts/unpack.sh`
5. Then run something like `./release/gotain run bash`

Have fun!

### Vagrant

```bash
vagrant up
vagrant ssh
cd /vagrant
./scripts/unpack.sh
./release/gotain run bash
```
